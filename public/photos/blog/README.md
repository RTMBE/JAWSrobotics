## Naming Convention:

Use descriptive names that indicate:
- What the image shows
- Which blog post it belongs to
- The date or version if applicable

Examples:
- `swerve-module-side-view.png`
- `swerve-module-top-view.png`
- `power-system-diagram.png`
- `wiring-harness-layout.png`

## Image Guidelines:

- Use PNG format for CAD screenshots (better quality)
- Use JPG format for photos
- Keep file sizes reasonable (under 2MB when possible)
- Use descriptive filenames (no spaces, use hyphens)
