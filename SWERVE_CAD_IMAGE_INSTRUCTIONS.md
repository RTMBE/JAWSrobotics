## How to Save the Images

1. **Save the first image (side cutaway view):**
   - Right-click on the image
   - Select "Save Image As..."
   - Navigate to: `public/photos/blog/`
   - Name it: `swerve-module-side-view.png`
   - Click Save

2. **Save the second image (top view):**
   - Right-click on the image
   - Select "Save Image As..."
   - Navigate to: `public/photos/blog/`
   - Name it: `swerve-module-top-view.png`
   - Click Save

## After Saving the Images

Once you've saved both images, run these commands:

```powershell
git add public/photos/blog/swerve-module-side-view.png
git add public/photos/blog/swerve-module-top-view.png
git commit -m "Add swerve module CAD images for Maksym's blog post"
git push
```
